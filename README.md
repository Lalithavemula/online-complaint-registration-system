# Resolvify Connect - Online Complaint Registration System

## 📌 Overview

**Resolvify Connect** is a modern online complaint registration and management system designed to simplify the process of submitting, tracking, and resolving complaints. The platform provides a user-friendly interface for citizens/users to register complaints and enables administrators to efficiently manage, monitor, and resolve issues.

The system improves transparency, communication, and efficiency by providing complaint tracking, status updates, notifications, and administrative management features.

---

## 🚀 Features

### 👤 User Features
- User registration and secure authentication
- Login/logout functionality
- Submit new complaints
- View complaint history
- Track complaint status
- Receive updates and notifications
- Manage user profile

### 🛠️ Admin Features
- Admin authentication
- View and manage all complaints
- Update complaint status
- Monitor complaint statistics
- Manage users
- Dashboard with analytics

## ⚙️ Installation and Setup

### Clone the repository

```bash
git clone https://github.com/Lalithavemula/online-complaint-registration-system.git
```

### Navigate to project directory

```bash
cd resolvify-connect-main
```

### Install dependencies

```bash
npm install
```

### Run the development server

```bash
npm run dev
```

The application will run at:

```
http://localhost:8080
```

---

## 🔐 Environment Variables

Create a `.env` file in the root directory and add required configuration:

```
VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_ANON_KEY=your_supabase_key
```
